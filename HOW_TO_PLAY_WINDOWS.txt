====================================================
  SAMSARA: Journey to the Triple Gem
  HOW TO PLAY ON WINDOWS
====================================================

GETTING STARTED
----------------
1. Unzip the downloaded file
2. Open the Samsara folder
3. Double-click Samsara.exe to launch the game

That's it — no Python, no installation, nothing else required.


CONTROLS
---------
  Move            : WASD or Arrow Keys
  Talk to NPC     : E
  Strike obstacle : X
  Meditate        : M
  Journal         : J
  Glossary        : G
  Help            : H
  Mute sound      : P
  Pause / Save    : ESC


TROUBLESHOOTING
----------------
"Windows protected your PC" warning on first launch?
  -> Click "More info" then "Run anyway"
  -> This appears because the game is not signed by Microsoft.
  -> It is safe to run.

Game won't open at all?
  -> Make sure you unzipped the folder first before running Samsara.exe
  -> Running the .exe directly from inside the zip will not work

No sound?
  -> Check your system volume is not muted
  -> Try pressing P in-game to toggle sound on/off
