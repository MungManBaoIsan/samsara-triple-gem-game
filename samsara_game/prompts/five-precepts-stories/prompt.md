# Five Precepts Stories

> **Category:** content-design
> **Model used:** claude-sonnet-4-6
> **Project area:** NPC dialogue and Buddhist ethics education
> **Status:** production
> **Last updated:** 2025-11-30

## What this prompt does

Adds five NPCs whose emotionally immersive stories teach the five core Buddhist ethical precepts — non-killing, non-stealing, sexual ethics, no intoxicants, and truthful speech — in language appropriate for both children and adults.

## The prompt

```
The dialogues should touch on player's emotions making the game feel really
immersive. Can you add a story about:
1. Killing
2. Stealing
3. Sexual misconduct
4. Intoxicants
5. False speech or lying
(5 precepts). Make it appropriate for children who might play but real
enough for adults to relate.
```

## Inputs

No variables — the five precept topics are specified directly.

## Expected output

Five new NPC characters with 3-dialogue transformation arcs each. Each story: names a specific, emotionally vivid incident (not an abstract moral); shows real consequences (lost trust, guilt, fear); leads to a genuine change in behaviour; and uses age-appropriate language throughout (no graphic violence, no explicit content). Each NPC is placed in a realm that matches the precept's associated mental state (e.g. stealing → Hungry Ghost realm / craving).
