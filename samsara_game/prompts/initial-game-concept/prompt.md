> **Last updated:** 2025-11-28

## What this prompt does

Describes the complete vision for a 2D Buddhist educational game set in samsara, establishing the six-realm world, three-currency progression system, and educational purpose for the entire project.

## The prompt

```
I would like if possible, to build a 2D game where the player is born into
samsara (a world of suffering). The aim of the game is to navigate your way
through this world full of difficulties such as being bullied, rejected,
losing loved ones, alcoholism, drug abuse, prostitution etc. People are full
of ignorance, delusion, hatred, greed, envy and other defilements. The world
is full of evil. The only place that is safe & peaceful is the triple gem at
the sacred temple. Eventually, the player who reaches the triple gem (ultimate
refuge) first wins.
```

## Inputs

No variables — this is a standalone concept brief delivered as a natural language description.

## Expected output

A complete Game Design Document including: world structure (six realms as concentric circles), gameplay loop, three-currency progression system (Merit, Karma, Wisdom), visual style recommendation, and a first phase implementation plan with working Python/Pygame code.
